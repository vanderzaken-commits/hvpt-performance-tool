# MVP Scope — Eerste Versie HVPT Performance Tool

## MVP-doel

De eerste versie van de HVPT Performance Tool moet bruikbaar zijn voor Harry als coach om hypertrofie/bodybuilding cliënten te begeleiden.

## Doelgroep

1. Harry als coach.
2. Cliënten die een hypertrofie/bodybuilding programma volgen.
3. Later uitbreidbaar naar algemene fitness, afvallen, kracht, sportprestatie en revalidatie.

## Must-have features

1. Coach-dashboard.
2. Cliëntprofiel.
3. Intakebasisgegevens.
4. Oefenbibliotheek.
5. Trainingsschema-builder.
6. Trainingslogboek.
7. 4 tot 13 weken sprint/programmaperiode.
8. Sets, reps, gewicht, rust, tempo, RIR en RPE.
9. Volume per spiergroep.
10. Progressie per oefening.
11. Basis cliëntomgeving.

## Should-have features

1. Filteren op spiergroep.
2. Filteren op materiaal.
3. Filteren op compound/isolatie.
4. Automatische waarschuwing bij te hoog of te laag volume.
5. Weekoverzicht.
6. Eenvoudige grafieken.

## Could-have features

1. PDF-export van schema.
2. Oefenvideo-link.
3. Coachnotities per oefening.
4. Client feedback na training.
5. Readiness-score.

## Niet in scope voor MVP

1. Voedingsschema’s.
2. Revalidatieprotocollen.
3. Facturatie.
4. Contractbeheer.
5. Volledige agenda.
6. Automatische app-notificaties.
7. Wearable-integratie.
8. AI-gegenereerde schema’s zonder coachcontrole.

## Shared core features

1. Gebruikersbeheer.
2. Cliëntbeheer.
3. Oefendatabase.
4. Programmastructuur.
5. Logging.
6. Analyse.
7. Dashboard.

## Hypertrofie-specifieke features

1. Spiergroepvolume.
2. Sets per spiergroep per week.
3. RIR-sturing.
4. RPE-monitoring.
5. Progressieve overload.
6. Deload-optie.
7. Oefenrotatie per sprint.
8. Mechanische spanning en trainingsvolume als basisprincipes.

## Definitief MVP-advies

Start klein maar professioneel.

Bouw eerst een betrouwbare hypertrofie/bodybuilding module met coach-dashboard, cliëntbeheer, schema-builder, logboek en basisanalyse.

Daarna pas uitbreiden.
