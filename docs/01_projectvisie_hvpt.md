# Projectvisie — HVPT Performance Tool

## Doel van de tool

De HVPT Performance Tool is een professionele coachingapp voor personal training, hypertrofie/bodybuilding, intake, trainingsschema’s, trainingslogging, voortgangsanalyse en later voeding, revalidatie en lifestyle coaching.

De tool is bedoeld om het werkproces van HVPT Performance te digitaliseren en te professionaliseren.

## Hoofddoel

Een modulair systeem bouwen waarin Harry als coach cliënten kan aanmaken, intakegegevens kan vastleggen, trainingsschema’s kan programmeren, prestaties kan monitoren en beslissingen kan nemen op basis van data.

## Eerste focus

De eerste versie richt zich uitsluitend op de hypertrofie/bodybuilding module.

## Belangrijke uitgangspunten

1. Evidence-based training.
2. Praktisch bruikbaar voor personal training.
3. Coach en cliënt krijgen ieder hun eigen omgeving.
4. De tool moet overzichtelijk, professioneel en schaalbaar zijn.
5. Codex mag alleen bouwen op basis van goedgekeurde projectdocumenten.

## Niet direct bouwen

- Voedingsmodule
- Revalidatiemodule
- Facturatie
- Volledige agenda-integratie
- Wearable-integratie
- Automatische AI-programmering zonder coachcontrole

Deze onderdelen kunnen later worden toegevoegd.
